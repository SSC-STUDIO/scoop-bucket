# SSC-STUDIO Scoop Bucket

Custom Scoop bucket for SSC-STUDIO Windows packages.

## Add the bucket

```powershell
scoop bucket add ssc-studio https://github.com/SSC-STUDIO/scoop-bucket
```

## Install Lenovo Legion Toolkit

```powershell
scoop install ssc-studio/lenovolegiontoolkit
```

## Update

```powershell
scoop update ssc-studio/lenovolegiontoolkit
```

## Remove

```powershell
scoop uninstall lenovolegiontoolkit
```

## Packages

- `lenovolegiontoolkit`
